# Ticker-车票儿 微信小程序重构版

## 项目概述

这是一个专业的车票生成工具微信小程序，支持多种车票样式，提供实时预览功能。

## 新版本特性

### 🎨 界面优化
- **更大的输入框**：字体大小从28rpx增加到32rpx，输入框高度增加到100rpx
- **更清晰的背景**：修复了背景渐变色过深的问题，使用更柔和的渐变
- **更好的视觉效果**：优化了按钮、卡片等组件的视觉效果

### 🏗️ 页面结构重构
- **首页**：项目介绍和车票样式卡片展示
- **制作页面**：实时预览、固定预览窗格、可滑动输入区域
- **历史结果页面**：云端同步、按日期排序、卡片展示
- **用户中心页面**：用户信息、小程序关于等

### ⚡ 功能增强
- **实时预览**：用户输入内容时自动生成预览，无需手动点击
- **固定预览窗格**：预览区域固定在页面顶部，输入区域可滚动
- **智能数据处理**：移除智能默认值，用户未填写字段返回空值
- **历史记录管理**：支持查看、重做、删除历史记录
- **云端同步**：历史记录自动保存到本地存储

## 使用说明

### 1. 首页
- 查看项目介绍和功能特色
- 浏览所有可用的车票样式
- 点击样式卡片直接跳转到制作页面

### 2. 制作页面
- 选择车票样式
- 填写车票信息（支持字段启用/禁用）
- 实时预览生成的车票
- 保存到相册并同步到历史记录

### 3. 历史结果页面
- 查看所有历史生成的车票
- 按样式筛选历史记录
- 重新制作或删除历史记录
- 导出历史记录

### 4. 用户中心页面
- 查看用户信息和统计数据
- 管理应用设置
- 清除缓存、检查更新
- 查看关于信息和隐私政策

## 技术特性

- **响应式设计**：适配不同屏幕尺寸
- **实时预览**：输入内容自动生成预览
- **本地存储**：草稿和历史记录本地保存
- **错误处理**：完善的错误处理和用户提示
- **性能优化**：防抖处理、懒加载等优化

## 注意事项

1. **图标文件**：需要在 `images/icons/` 目录下添加实际的图标文件
2. **预览图片**：需要在 `static/previews/` 目录下添加车票样式预览图片
3. **API配置**：确保API服务器正常运行，URL配置正确
4. **权限设置**：需要相册权限才能保存图片到相册
